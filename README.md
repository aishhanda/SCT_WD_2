# ⏱️ Stopwatch Web Application

An interactive and user-friendly stopwatch web application built as part of my **Web Development Internship at Skillcraft Technology**.

The application allows users to accurately measure time intervals with support for lap tracking, theme switching, and a responsive modern UI.

---

## 🚀 Features

- Start, Pause, and Reset functionality
- Lap time recording with millisecond precision
- Dark / Light theme toggle 🌗
- Smooth animations and glassmorphism UI
- Fully responsive (mobile, tablet, desktop)
- Clean and maintainable code structure

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (CSS Variables, Flexbox, Media Queries)
- JavaScript (DOM manipulation, timing logic)
